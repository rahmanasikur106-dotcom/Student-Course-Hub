# Nexus Institute: Student_Course_Hub

## Project Overview
This web application is designed to manage academic programmes, staff profiles, and student interests for the Nexus Institute. It demonstrates a full-stack approach using the LAMP stack (Linux/Apache/MySQL/PHP).

## Key Features
* **Admin Dashboard:** Full CRUD (Create, Read, Update, Delete) functionality for managing programmes and staff.
* **Student Interface:** A modern, high-fidelity UI allowing students to browse courses and view detailed academic bios.
* **Database Integration:** Relational database design with normalized tables for Programmes, Levels, and Staff.
* **Security:** Implementation of PHP Prepared Statements to prevent SQL Injection.

## Technologies Used
* **Backend:** PHP 8.x
* **Database:** MySQL
* **Frontend:** HTML5, CSS3 (Custom Glassmorphism Design)
* **Tools:** XAMPP, Git, GitHub
